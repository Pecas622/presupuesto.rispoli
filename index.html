Plataforma LMS para Go Travel Academy

Desarrollo de un sistema de capacitación online personalizado: cursos, alumnos, pagos y seguimiento del aprendizaje en una sola plataforma.

01

Objetivo

Desarrollar una plataforma de capacitación online (LMS) personalizada para Go Travel Academy, que permita la gestión integral de cursos, usuarios, pagos y seguimiento del aprendizaje, con una arquitectura escalable y moderna.

02

Funcionalidades

Tres módulos integrados que cubren todo el ciclo: descubrimiento público, experiencia del alumno y control administrativo.

Plataforma pública

  • Página de inicio
  • Catálogo de cursos
  • Buscador y filtros
  • Detalle de cada curso
  • Registro e inicio de sesión
  • Recuperación de contraseña
  • Perfil del alumno

Plataforma del alumno

  • Acceso a cursos adquiridos
  • Reproductor de videos
  • Material descargable
  • Evaluaciones
  • Seguimiento de progreso
  • Certificados de finalización
  • Historial de cursos

Panel administrativo

  • Gestión de cursos
  • Gestión de módulos y clases
  • Gestión de alumnos
  • Gestión de instructores
  • Reportes
  • Estadísticas
  • Gestión de pagos
  • Configuración general

Seguridad

  • Usuarios con distintos permisos
  • Protección de datos
  • Copias de seguridad
  • Accesos seguros
03

Pagos

Cobro integrado y automático según el origen del alumno, con selección de moneda transparente.

Mercado Pago

  • Clientes argentinos
  • Pago en cuotas

Stripe

  • Clientes internacionales

Moneda automática

  • Selección automática ARS / USD según el usuario
04

Stack moderno, escalable y con soporte a largo plazo.

React Node.js Express PostgreSQL SQL Cloud Storage API Mercado Pago API Stripe
