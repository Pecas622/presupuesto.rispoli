Plataforma LMS
para Rispoli

Desarrollo de un sistema de capacitación online personalizado: cursos, alumnos, pagos y seguimiento del aprendizaje en una sola plataforma.

ClienteRispoli
DesarrolladorSantiago Zorrilla
Fecha30 de junio de 2026
Validez15 días

01Objetivo

Desarrollar una plataforma de capacitación online (LMS) personalizada para Rispoli, que permita la gestión integral de cursos, usuarios, pagos y seguimiento del aprendizaje, con una arquitectura escalable y moderna.

02Funcionalidades

Tres módulos integrados que cubren todo el ciclo: descubrimiento público, experiencia del alumno y control administrativo.

Plataforma pública

  • Página de inicio
  • Catálogo de cursos
  • Buscador y filtros
  • Detalle de cada curso
  • Registro e inicio de sesión
  • Recuperación de contraseña
  • Perfil del alumno

Plataforma del alumno

  • Acceso a cursos adquiridos
  • Reproductor de videos
  • Material descargable
  • Evaluaciones
  • Seguimiento de progreso
  • Certificados de finalización
  • Historial de cursos

Panel administrativo

  • Gestión de cursos
  • Gestión de módulos y clases
  • Gestión de alumnos
  • Gestión de instructores
  • Reportes
  • Estadísticas
  • Gestión de pagos
  • Configuración general

Seguridad

  • Usuarios con distintos permisos
  • Protección de datos
  • Copias de seguridad
  • Accesos seguros

03Pagos

Cobro integrado y automático según el origen del alumno, con selección de moneda transparente.

Mercado Pago

  • Clientes argentinos
  • Pago en cuotas

Stripe

  • Clientes internacionales

Moneda automática

  • Selección automática ARS / USD según el usuario

04Tecnologías

Stack moderno, escalable y con soporte a largo plazo.

React
Node.js
Express
PostgreSQL
SQL
Cloud Storage
API Mercado Pago
API Stripe

05Tiempo estimado

10–14 semanas

Sujeto a la entrega del contenido por parte del cliente.
